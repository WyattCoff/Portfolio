This project was created at TafeSA under the 'Write scripts for software applications' course. This website simulates a customer check out. The user is promped with to enter there customer details (first name, last name, email, postcode) and there payment method (credit card (visa or mastercar), card number, cvv and expiry date). Each feild has appropriate validation. The user has 3 buttons below the form. The first is a submit button, which submits the form. The next button is a help button which displays a pop up window which explains to the user the requirements for each feild. and a 'Go to Other Websites' button which once pressed promps the user to go to Google.com, tafesa.edu.au or tafesa.edu.au/mytafe-sa. This website rises to the screen being used so it is usable on any device.
